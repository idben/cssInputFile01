# 客制化上傳檔案按鈕的基本範例

## 專案摘要

1. 讓檔案上傳的按鈕不再只是系統的預設樣式
2. 每一組樣式都需要搭配 css 樣式及 javascript 的動作才能有完整行為。

## 操作
1. 建立專案資料夾後
2. 開啟終端機或命令提示字元
3. 不要 git init
4. 貼上以下指令，就可以直接在專案資料夾下下載專案，不會再多一層 
```bash
git clone git@github.com:idben/cssInputFile01.git .
```
